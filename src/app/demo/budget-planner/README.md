# Budget Planner Demo

A fully interactive demo of the MoneyPlan Budget Planner application.

## Features

### All Sections Implemented:
- ✅ **Dashboard** - Financial overview with insights
- ✅ **Income** - Track income sources
- ✅ **Expenses** - Manage expenses with categories
- ✅ **Savings** - Monitor savings goals
- ✅ **Debts** - Track debt payoff progress
- ✅ **Subscriptions** - Manage recurring subscriptions
- ✅ **Settings** - View preferences

## Interactive Capabilities

- Add new income, expenses, and savings entries
- Delete entries with real-time updates
- Search and filter transactions
- View progress bars for goals and debts
- Calculate financial metrics automatically
- Responsive design for all devices

## Sample Data

The demo includes realistic mock data:
- 3 income sources ($6,050 total)
- 6 expense entries ($2,300 total)
- 3 savings goals ($1,200 total)
- 3 debt accounts ($27,900 remaining)
- 4 active subscriptions ($80.97/month)

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React Icons
- React Context API

## Usage

Navigate between sections using the sidebar. All data is stored in memory and resets on page refresh.
